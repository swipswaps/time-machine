## Features
* Downloads laurent22's rsync_tmbackup.sh


## Dependencies
### Arch
```
sudo pacman -S rsync xdg-utils
```
### Debian/Ubuntu/elementaryOS
```
sudo apt install rsync xdg-utils
```

## Run the script
```
./time-machine
```
